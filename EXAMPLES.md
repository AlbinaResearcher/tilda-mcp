## Full Workflows

### Build a portfolio site from scratch
```
Help me create a portfolio website:
1. Create a new project called "MyPortfolio2024"
2. Add pages: Home, Work, About, Contact
3. Add a heading "My Portfolio" to the Home page
4. Add an image gallery to the Work page
5. Add a contact form to the Contact page
6. Publish the project when everything is ready
```

### Update an existing site
```
In project 12345:
1. Show me all pages
2. Update the text on the Home page
3. Add a news section
4. Publish the changes
```

### Create a landing page
```
Create a new project "ProductLaunch" with a landing page that includes:
- A hero section with a headline and description
- A list of key benefits
- An email signup form
- A "Buy Now" button
Then publish it.
```

---

## Tips

- You can give Claude multi-step instructions and it will execute them in order
- Use natural language — Claude understands intent, not just commands
- Always double-check before publishing: ask Claude to summarize what it has done
- If something goes wrong, ask Claude: "What was the last thing you changed?"
