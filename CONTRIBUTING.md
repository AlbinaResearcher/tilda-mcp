# Contributing

Thank you for your interest in contributing to Tilda MCP!

## How to contribute

1. **Fork** this repository
2. **Create a branch** for your change: `git checkout -b feature/my-improvement`
3. **Make your changes**
4. **Commit** with a clear message: `git commit -m "Add: description of change"`
5. **Push** your branch: `git push origin feature/my-improvement`
6. **Open a Pull Request** — describe what you changed and why

## What we welcome

- Bug fixes
- Documentation improvements
- New usage examples
- Setup guides for additional platforms

## Reporting issues

Open an [issue](../../issues) and include:
- Your OS and Node.js version
- What you expected to happen
- What actually happened
- Any error messages

## Code style

Keep it simple and readable. No unnecessary abstractions.
